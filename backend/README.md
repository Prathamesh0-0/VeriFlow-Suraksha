# VeriFlow Backend API

This is the backend API for the VeriFlow document forensics engine.
It is built using FastAPI and runs in a Docker container with OpenCV, PyMuPDF, and Tesseract OCR dependencies.

This repository is automatically synced from the main GitHub repository.
